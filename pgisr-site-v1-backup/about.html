The Company

PWI, Inc.

PWI, Inc. is an independent electronics manufacturer headquartered in Wichita, Kansas. Since 1963, PWI has designed and built precision electronic products for aerospace, industrial, and commercial security customers worldwide.

PGISR is PWI's persistent ground sensor platform — designed for private property, commercial buildings, and industrial facilities that need reliable, unattended perimeter coverage without a large on-site security footprint.

Every PGISR unit is designed, assembled, and tested in the USA. PWI maintains strict quality controls under ISO 9001 and AS9100.

What We Deliver

Built for Property Security

IP68 Enclosures

All sensor nodes are housed in impact-resistant, IP68-rated enclosures — fully sealed against dust and water immersion.

Encrypted Communications

End-to-end encrypted data transmission. Standard API integration allows connection to any existing building management or security platform.

GPS Built In

Every node includes integrated GPS so sensor locations are automatically reported with all detection events.

Multi-Sensor Nodes

Multiple sensor types can be combined into a single enclosure — reducing deployment footprint while broadening detection coverage.

AI Threat Identification

On-node AI processing reduces false positives and enables real-time threat classification before data leaves the sensor.

Rechargeable & Field-Ready

Extended rechargeable battery life supports continuous unattended operation — ideal for perimeter locations without convenient power access.
